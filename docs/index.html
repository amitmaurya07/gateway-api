Introduction

Service APIs is the evolution of Kubernetes APIs that relate to Services, e.g. Ingress. The project is part of Kubernetes, working under SIG-NETWORK. Use the following resources to learn more about Service APIs.

Get started

To get started, please read through the API overview and Security model. These documents give the necessary background to understand the API and use-cases it targets.

Guides

Once you have a good understanding of the API at a higher-level, please follow one of our guides to dive deeper into different parts of the API.
